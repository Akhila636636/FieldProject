# **App Name**: DevPath

## Core Features:

- User Chat Interface: An intuitive chat input box for users to describe their interests and questions.
- AI Interest Profiler: Utilizes the Gemini API as a tool to analyze user chat input and understand their technical interests, skills, and experience.
- AI Project Recommendation Engine: Leverages the Gemini API as a tool to generate and provide 3 personalized, beginner-friendly project ideas with brief descriptions based on the profiled user interests.
- User Profile Summary Display: Nicely displays a simple summary or 'user profile' inferred by the AI from the user's input (e.g., 'Interested in Web Dev, JavaScript').
- Project Recommendations Display: Presents the 3 AI-generated project titles and descriptions clearly and legibly for the user.
- Backend API Communication Layer: Manages the relaying of user chat input to the Gemini API and handles the return of AI responses to the frontend.

## Style Guidelines:

- Primary color: A deep, thoughtful blue-violet (#442966) that signifies focus and the creative journey of development, balanced for a light color scheme.
- Background color: A very light, desaturated version of the primary hue (#F4F3F5) to ensure clarity and provide a clean canvas for content.
- Accent color: A vibrant sky blue (#2C89E6) to provide clear visual cues, highlight interactive elements, and add an encouraging contrast.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring excellent readability for chat, profiles, and project descriptions.
- Utilize simple, line-art or minimalist solid icons throughout the application, representing concepts such as chat, learning, project ideas, and navigational cues to maintain a clean and uncluttered interface.
- Implement a clean and functional layout, possibly with the chat input fixed at the bottom and results displayed in a scrollable area above, ensuring optimal use of screen space and clear content hierarchy.
- Incorporate subtle and quick animations for feedback, such as a brief loading indicator when sending messages or receiving AI recommendations, and gentle fades for new content appearing on the screen to enhance user experience without distraction.
